GURU Ai

Installation

git clone https://github.com/Guru322/GURU-Ai.git
cd GURU-Ai
npm install

Configuration

Create a .env file or set these environment variables:

MONGODB_URI=<Your MongoDB connection URI>
PHONE_NUMBER=<Your WhatsApp phone number>
BOTNAME=<Bot display name>
OWNERS="<owner1;john;owner2;doe>"

Running the Bot

npm start
# or
node index.js

Usage

.ping
.menu
.list
.alive
